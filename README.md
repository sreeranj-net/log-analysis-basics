# Log Analysis Basics (Linux)

This project demonstrates basic security log analysis skills using Linux authentication logs.  
The focus is on identifying unauthorized SSH login attempts and understanding common attack patterns.

## Project Overview
- Collected SSH authentication logs from a Linux system
- Identified failed login attempts for invalid users
- Analyzed patterns indicating possible brute-force behavior
- Documented findings in an incident report

## Tools & Skills Used
- Linux (Kali)
- journalctl
- grep
- Basic SOC analysis
- Incident documentation

## Files Included
- `sample-logs/auth.log` – SSH authentication logs
- `commands-used.md` – Commands used during analysis
- `incident-report.md` – Security incident summary

## Disclaimer
Logs are collected from a lab environment and are used for learning and demonstration purposes only.
